## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Gunal-k/MERN-Stack-Projects
```

Navigate to the project folder:

```bash
cd MERN-Stack-Projects
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
